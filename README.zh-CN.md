goose 是一款运行在你本地机器上的通用 AI 助手。不仅限于代码——可用于研究、写作、自动化、数据分析或任何你需要完成的工作。

面向 macOS、Linux 和 Windows 的原生桌面应用。完整的 CLI 用于终端工作流。API 可嵌入任何地方。采用 Rust 构建，兼具高性能和可移植性。

goose 支持 15+ 提供商——Anthropic、OpenAI、Google、Ollama、OpenRouter、Azure、Bedrock 等。可通过 [ACP](https://goose-docs.ai/docs/guides/acp-providers) 使用 API 密钥或你现有的 Claude、ChatGPT 或 Gemini 订阅。通过 [Model Context Protocol](https://modelcontextprotocol.io/) 开放标准可连接 70+ 扩展。

goose 是 Linux 基金会旗下 [Agentic AI Foundation (AAIF)](https://aaif.io/) 的一部分。

# 快速开始

**[下载桌面应用](https://goose-docs.ai/docs/getting-started/installation)**，支持 macOS、Linux 和 Windows。

或安装 CLI：

```bash
curl -fsSL https://github.com/aaif-goose/goose/releases/download/stable/download_cli.sh | bash
```
