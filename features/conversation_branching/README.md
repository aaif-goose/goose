# Conversation Branching

Fork any Goose session at a specific message and continue independently.

## How to Run

### CLI Mode
```bash
python3 branch_session.py --interactive
```

### Web UI Mode
```bash
streamlit run branch_ui.py
```

## Features
- List all sessions and messages
- Branch from any message point
- Branched session appears automatically in Goose sidebar
- Interactive CLI mode
- Visual Streamlit web UI

## Requirements
```bash
pip install streamlit
```
